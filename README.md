## PPP-board

#### 스프링부트 연습용 간단한 게시판 만들기
- 유저 관리
  - 회원가입
  - 회원 정보 수정
  - 회원 정보 조회
  - 로그인
  - 로그아웃
    
- 게시글 관리
  - 게시글 리스트
  - 게시글 검색
  - 게시글 생성
  - 게시글 수정
  - 게시글 삭제

- 댓글 관리
  - 댓글 생성
  - 댓글 수정
  - 댓글 삭제
  - 댓글 조회 (게시글 기준)

- 첨부 파일 관리
  - 첨부 파일 추가
  - 첨부 파일 삭제
  - 순서 업데이트
